## API Config: strumento di configurazione del Nodo dei Pagamenti ##

La piattaforma offre degli strumenti di gestione delle configurazioni del Nodo dei Pagamenti.

### Dominio EC ###
Dominio EC raggruppa le sezioni riguardanti gli Enti Creditori (EC).
In particolare:
- **EC**: visualizzazione, creazione, modifica e cancellazione di un Ente Creditore.
- **Intermediari EC**: visualizzazione, creazione, modifica e cancellazione di un intermediario di un Ente Creditore.
- **Stazioni**: visualizzazione, creazione, modifica e cancellazione di una stazione.
- **Informativa Conto Accredito**: download, upload e cancellazione di un XML riguardante l'ICA.
  - **Verifica Informativa Conto Accredito**: effettua dei controlli riguardanti la correttezza di un file ICA (si rimanda al [link](https://pagopa.atlassian.net/wiki/spaces/PAG/pages/441517396/Verifica+ICA) per gli approfondimenti sui controlli sintattici e semantici effettuati).
- **Tabella delle Controparti**: download, upload e cancellazione di un XML riguardante la Tabella delle Controparti (si rimanda al [link](https://pagopa.atlassian.net/wiki/spaces/PAG/pages/466093190/Verifica+Tabella+delle+controparti) per gli approfondimenti sui controlli sintattici e semantici effettuati).

### Dominio PSP ###
Dominio PSP raggruppa le sezioni riguardanti i Prestatori di Servizio di Pagamento (PSP).
In particolare:
- **PSP**: visualizzazione, creazione, modifica e cancellazione di un Prestatore di Servizio di Pagamento.
- **Intermediari PSP**: visualizzazione, creazione, modifica e cancellazione di un intermediario PSP.
- **Canali**: visualizzazione, creazione, modifica e cancellazione di un Canale.
- **Catalogo Dati Informativi**: download, upload e cancellazione di un XML riguardante il Catalogo Dati Informativi (si rimanda al [link](https://pagopa.atlassian.net/wiki/spaces/PAG/pages/467435579/Verifica+CDI) per gli approfondimenti sui controlli sintattici e semantici effettuati).

### Configuration ###
Configuration raggruppa le sezioni riguardanti le configurazioni di funzionamento del Nodo dei Pagamenti.
In particolare:
- **Configuration Keys**: visualizzazione, creazione, modifica e cancellazione di una configurazione del Nodo dei Pagamenti.
- **PDD**: visualizzazione, creazione, modifica e cancellazione di una configurazione delle Porte di Dominio.
- **WFESP Plugins**: visualizzazione, creazione, modifica e cancellazione di una configurazione dei plugin WFESP.
